# Autopilot v0.1.0

## Downloads

| Platform | File |
|----------|------|
| Windows x64 | `autopilot-win32-x64.zip` |
| macOS ARM64 | `autopilot-darwin-arm64.zip` |
| macOS x64 | `autopilot-darwin-x64.zip` |
| Linux x64 | `autopilot-linux-x64.tar.gz` |

## How to Use

Extract the archive for your platform and run the application:

- **Windows:** Unzip, then double-click `Autopilot.exe`
- **macOS:** Unzip, then open `Autopilot.app`
- **Linux:** `tar -xzf autopilot-linux-x64.tar.gz && ./autopilot`

> **Note:** This repo contains large build artifacts (~1.4 GB). Cloning the full repo will be larger than a source-only clone. Old version folders are removed when new ones are added.
