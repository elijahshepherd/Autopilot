v0.9.0 Highlights
• Reasoning is private — OpenAI reasoning_content and Anthropic thinking_delta stream as a collapsed thinking disclosure; never leak into the visible message text
• Malformed tool-call scrubber — if a model emits raw JSON like {"questions": [...]} or a tool envelope, it's stripped before reaching the UI
• Bounded rate-limit handling — at most one retry (30-second ceiling), then a clear user-facing message; no infinite loops or runaway waiting animations
• Capability manifest — each request carries a startup system message listing enabled tools and platform/elevation so the AI knows what's available
• Streaming hardened — provider now uses LanguageModelResponsePart2, matching the BYOK Anthropic/Gemini providers

How to use
Windows: Extract the zip, double-click Autopilot.exe (UAC prompt will appear)
macOS: Extract the zip, open Autopilot.app
Linux: tar -xzf autopilot-linux-x64.tar.gz && ./autopilot

Models
Ollama auto-detected on localhost:11434. Set OPENAI_API_KEY and OPENAI_BASE_URL (defaults to https://api.openai.com/v1) for any OpenAI-compatible endpoint including NVIDIA NIM. Set ANTHROPIC_API_KEY for Anthropic. Or drop a JSON config at ~/.autopilot/providers.json.

These are unsigned portable builds. Windows SmartScreen may flag the executable.
