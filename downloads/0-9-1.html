v0.9.1 Highlights
• Fliers in chat input — click the "Flier" entry in the chat input chip area, or run Autopilot: Set Autopilot flier, to switch between built-in Fliers (e.g. roblox) and your custom .autopilot/fliers/.md files
• Plan chat mode added to the Ask/Edit/Agent pills
• Underlying error surfaced when chat forwarding fails (mentions, etc.) instead of a generic "Failed to get a response" notice
• Sentence-case UI across the chat-status dashboard, chat setup, and the agents picker
• Auth menu toggleable — the Accounts icon returns as a user-toggleable command
• Friendly rate-limit copy in the chat status widget

How to use
Windows: Extract the zip, double-click Autopilot.exe (UAC prompt will appear)
macOS: Extract the zip, open Autopilot.app
Linux: tar -xzf autopilot-linux-x64.tar.gz && ./autopilot
