v0.9.2 Highlights
• Editor pane in Agents Window — the main editor area is now visible alongside the chat bar, enabling seamless code editing during agent sessions
• Separate Agents.exe bundle — dedicated Agents.exe with admin launcher for elevated sessions (run Start-Process -Verb RunAs Autopilot.exe --agents)
• MiniMax M3 on NVIDIA NIM fixed — BYOK providers now support includeReasoningContentInHistory: false to disable reasoning_content in assistant history, preventing 500 errors on endpoints that don't support it (e.g., NVIDIA NIM)
• Admin startup hang fixed — welcome setup service now has 5-second timeouts on profile/account operations to prevent hanging when running as admin with different profile paths
• Sentence-case UI — continued cleanup of title-cased labels across chat setup, agents picker, and status widgets

How to use
Windows: Extract the zip, double-click Autopilot.exe (UAC prompt will appear). For admin agents: Start-Process -Verb RunAs Autopilot.exe --agents
macOS: Extract the zip, open Autopilot.app
Linux: tar -xzf autopilot-linux-x64.tar.gz && ./autopilot
