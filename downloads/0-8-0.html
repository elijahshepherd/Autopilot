v0.8.0 Highlights
• Agents Window: separate admin EXE — spawns a fully elevated window in its own process instead of opening in the editor window (which used to "load forever")
• Debug chat mode — new "Debug" pill in the chat header for autonomous diagnose-and-fix workflows
• AI identity — chat responses now say "Autopilot" with a direct, easy-to-understand tone (no emojis)
• CAPI fallback — #codebase and endpoint resolution now fall back to local models when CAPI fails
• Status bar fully hidden — Copilot quota/status icons never appear
• MCP servers removed — vscode-automation-mcp and component-explorer entries cleaned out

How to use
Windows: Extract the zip, double-click Autopilot.exe (UAC prompt will appear)
macOS: Extract the zip, open Autopilot.app
Linux: tar -xzf autopilot-linux-x64.tar.gz && ./autopilot

Models
Ollama auto-detected on localhost:11434. Set OPENAI_API_KEY and OPENAI_BASE_URL (defaults to https://api.openai.com/v1) for any OpenAI-compatible endpoint including NVIDIA NIM. Set ANTHROPIC_API_KEY for Anthropic. Or drop a JSON config at ~/.autopilot/providers.json.

These are unsigned portable builds. Windows SmartScreen may flag the executable.
