# Contributing to Autopilot

Welcome, and thank you for your interest in contributing to Autopilot!

Autopilot is an open-source editor focused on AI-assisted coding and computer tasks.

## Issue Reporting Only

We accept issue reports for two purposes only:

- **Bug reports**: Reproducible problems with Autopilot
- **Feature requests**: Ideas for improving Autopilot

We do **not** accept code contributions via pull request at this time.

### Before Filing an Issue

- Search existing issues to see if your bug or feature request already exists
- Try reproducing the issue after disabling all extensions to isolate the problem
- Provide clear, reproducible steps

### Filing a Good Bug Report

Include the following:

- Version of Autopilot
- Your operating system
- List of installed extensions
- Reproducible steps (1... 2... 3...)
- What you expected vs what actually happened
- Errors from the Dev Tools Console (Help > Toggle Developer Tools)
